本程序实现了需求的功能。
1.读取三个文件夹里的数据，以players中的为主角，和male以及female里的数据按照给定的规律进行配对。并且本客户端界面上显示有所有人的配对情况和匹配值以及专门显示了主角的配对情况和配对值。
2.可以让用户自己输入主角的各项参数，同时随机生成male1和female1两个文件，文件中的数据都是随机生成并且符合题目给定的规律。而且在界面也配有所有人的配对信息和主角单独的配对信息，方面查看主角的配对情况。




附件在attach文件夹中，而且给出了本程序的源程序代码供查看。


运行只需要点击attach.bat文件即可，会自动调用attach.jar文件。当然，前提条件是运行的客户机必须事先装有jdk。


提示：如果发现打开客户端，按钮和输入框不全的话，请点击最大化。
