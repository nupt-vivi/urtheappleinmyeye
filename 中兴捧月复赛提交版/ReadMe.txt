**************************************************************************************
本程序实现了需求的功能。
1.读取了两个文件中的数据，并且按照题目给定的方法进行配对。并且在客户端上面输出了男生和女生的配对情况。因为女生有一个最低的期望值，导致很多男生和女生无法配对。这和现实情况很符合。
2.可以让用户自己输入需要的男生数和女生数，同时随机生成maleRandom和femaleRandom两个文件，文件中的数据都是随机生成并且符合题目给定的规律。而且在界面也配有所有人的配对信息。
**************************************************************************************



**************************************************************************************
两个功能都和题目给定的相符合。添加了健康这个属性，并且取消了预赛中主角这个角色，直接从男生和女生中开始配对。女生从候选的男生中按照降序来进行选择，如果这一轮没有合适的人选，让收到邀请数仅次于这个女生的女生来进行选择配对，以此类推。
**************************************************************************************



**************************************************************************************
为了界面的美观，我对输出格式进行了格式化。使用四位的大小，不足的使用_来补上。
**************************************************************************************



**************************************************************************************
源代码中附有单元测试。
题目给定的文本附件以及用户随机产生的数据都在attach文件夹中，而且给出了本程序的源程序代码供查看。
**************************************************************************************
